# DIGITAL PROJECT


_Учебный проект сайта архитектурной организации._

Включает в себя:

1. Лендинг с формой обратной связи.
2. Галерея фотографий.
3. Страница с примерами проектов.
4. Сертификаты компании.
5. Контактная информация.
